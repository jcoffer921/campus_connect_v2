What is Campus Connect?

Campus Connect is a student networking platform built with Python and Django that uses Graphs to model student relationships and Trees to structure discussion threads. It helps students collaborate, find peers in similar majors, and engage in campus discussions.

Build Connections

Find peers based on shared interests or courses using intelligent graph algorithms.

Join Discussions

Engage in structured discussions organized hierarchically with tree structures.

Send Messages

Communicate privately with other students through the messaging system.
